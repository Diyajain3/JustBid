# 📌 JustBid

JustBid is a web application that helps users discover and manage public procurement opportunities in a simple and efficient way.

The project is currently in development.

# 🚀 Features (In Progress)
Centralized listing of procurement opportunities
Basic filtering and search system
Modern UI with responsive design
Dashboard for viewing opportunities


# 🛠 Tech Stack
React.js
Tailwind CSS
GSAP (animations)
React Router

# 📁 Project Status
⚠️ This project is currently under active development.
Features and UI may change frequently.
